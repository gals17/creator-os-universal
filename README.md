# Creator OS

Creator OS - универсальное приложение для целей, ежедневной практики, действий и журнала прогресса.

Подробное объяснение философии системы: [ABOUT-CREATOR-OS.md](ABOUT-CREATOR-OS.md)

## Что это дает

Приложение помогает человеку:

- выбрать одну главную цель;
- пройти практику сознания и выбрать состояние;
- задать срок и измеримый результат;
- вести ежедневную настройку состояния;
- создавать визуальный образ будущей версии себя;
- собирать промпты для ChatGPT Image 2 и Reve;
- загружать картинку-образ цели в приложение;
- проходить чек-лист перед важным действием;
- записывать факты прогресса голосом или текстом;
- раз в неделю делать разбор;
- установить систему на Android как приложение.

Главная идея:

**одна настоящая цель + нужное состояние + ежедневное действие + журнал фактов = управляемое движение к новой реальности.**

Creator OS построена на принципе одной цели, осознания сознания, выбора вибрации, фокуса внимания, входа в состояние будущей версии себя, ежедневного действия и честного разбора фактов.

## Для кого

Creator OS подходит для целей в разных сферах:

- деньги;
- бизнес;
- здоровье;
- тело;
- обучение;
- дисциплина;
- творчество;
- отношения;
- личная или духовная практика.

## Что человек получит через 30 дней

Если пользоваться приложением ежедневно, у человека появится:

- ясная формулировка цели;
- история выполненных действий;
- личный визуальный якорь цели;
- понимание, какие состояния помогают и мешают;
- список ошибок и повторяющихся паттернов;
- одно правило улучшения на каждую неделю;
- ощущение, что цель стала системой, а не просто желанием.

## Как пользоваться

1. Открой раздел **Цель**.
2. Выбери сферу: деньги, здоровье, бизнес, обучение, дисциплина, творчество и т.д.
3. Если цель денежная, выбери валюту: рубли, доллары или евро.
4. Запиши цель, текущий уровень, целевой уровень и срок.
5. Каждый день открывай **День** и проходи короткий протокол.
6. В разделе **Образ** опиши будущую сцену и собери промпт для генерации картинки.
7. Перед важным шагом открывай **Действие**.
8. Вечером заполняй **Журнал**.
9. Раз в неделю открывай **Разбор**.

## Установка на Android

1. Открой ссылку на приложение в Chrome на Android.
2. Нажми три точки справа сверху.
3. Выбери **Установить приложение** или **Добавить на главный экран**.
4. Подтверди.

После этого на телефоне появится иконка Creator OS.

Пользователю не нужен свой GitHub, чтобы установить приложение. Он открывает вашу опубликованную ссылку и устанавливает Creator OS на свой телефон. Его данные будут храниться отдельно на его устройстве.

## Установка на iPhone / iOS

1. Открой ссылку на приложение в **Safari**.
2. Нажми кнопку **Поделиться**.
3. Выбери **На экран Домой** или **Add to Home Screen**.
4. Нажми **Добавить**.

После этого на экране iPhone появится иконка Creator OS.

## Как делиться приложением

После публикации на GitHub Pages вы даете человеку ссылку вида:

`https://ТВОЙ-ЛОГИН.github.io/creator-os-universal/`

Человек открывает эту ссылку на телефоне и устанавливает приложение. Свой GitHub ему не нужен.

## Приватность

Данные пользователя сохраняются локально в браузере на его устройстве. Если другой человек открывает ту же ссылку, он получает чистое приложение со своими данными.

## Публикация на GitHub Pages

1. Создай публичный репозиторий на GitHub, например `creator-os`.
2. Загрузи файлы из этой папки в корень репозитория:
   - `index.html`
   - `manifest.webmanifest`
   - `sw.js`
   - `icon.svg`
   - `README.md`
3. Открой **Settings -> Pages**.
4. Source: **Deploy from a branch**.
5. Branch: **main**, Folder: **/root**.
6. Нажми **Save**.

Через 1-3 минуты сайт будет доступен по ссылке:

`https://ТВОЙ-ЛОГИН.github.io/creator-os/`
